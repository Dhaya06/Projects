## CSI-FORM

CSI-FORM is use to generate form dynamically.

## DEPENDANCIES

csi-page-layout

## USAGE

<csi-form 
    [csiFormGroup]="<form-group>"
    [csiFormFields]="<form-form-fields>"
    [csiFormLayout]="<form-layout>"
    [csiFieldEvents]="<form-imput-element-events>"
    [csiComponentId]="<component-id>"
    (csiSubmit)="<form-submit()>"
    ></csi-form>
